Nasa App

Description

The NASA App is an app that uses the NASA API. This is the final project made for the treehouse degree. It has two functions which are called Rover Postcard Maker and Eye-In-The-Sky. Rover Postcard Maker is where the user chooses an image from the Mars Rover Imagery API and adds a text over the selected image. The user then can send it through a prepopulated email address. Eye-In-The-Sky is a tool that accesses the Earth Imagery API. Users are able to search a location and obtain the most recent photo of the particular location on earth.

Tools used:

UICollectionViews, ScrollViews, Animation, Unit Testing, Error Handling, Asynchronous Networking
